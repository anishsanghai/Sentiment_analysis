Sentiment Analysis of the BlackCoffer Articles

The following folder contains the project which has the following files:-
input.xlsx == this file contains the url id's of the various articles whose data has to be extracted and analysis needs to be done.
Stopwords == This folder contains the various stopwords text documents that has to be used in the analysis.
MasterDictionary ==  This folder contains the the positive and negative words text document that are to be used for calculating the scores.
extracted_files == This folder contains the articles that has been extracted by the program data_extraction.py.
data_extraction.py == This program has the code for extracting the content from the articles given in the input.xlsx.
analysis.py == This program contains the pre_processing, sentiment analysis and writing the output back into the excel code.
output.xlsx == This excel file has the result or the output of this sentiment analysis
